This Power BI dashboard provides a comprehensive analysis of loan and financial data to support data-driven decision-making. The dashboard highlights key financial metrics, loan performance indicators, and risk segmentation to help stakeholders understand overall portfolio health.

Key Features:

Total loans, average loan amount, and loan distribution analysis

Risk-level categorization (Low, Medium, High) with comparative insights

Charge-off analysis to identify default trends

Interactive filters for dynamic exploration of loan data

Clean and user-friendly visualizations for quick insights

Tools & Technologies:

Power BI

SQL (for data extraction and transformation)

Excel (data preprocessing)

Outcome:
Enabled better understanding of loan trends, risk exposure, and financial performance, supporting informed business and risk management decisions.
